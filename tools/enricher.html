¿Cómo funciona?

1. Descarga los JSONs desde GitHub directamente.

2. Para cada película, llama a la API de TMDB y obtiene director, año, sinopsis y poster.

3. Solo completa campos vacíos — no sobreescribe datos existentes.

4. Descargá los JSONs enriquecidos y subílos al repo (arrastrando en GitHub.com).
